# L-Ease

L-Ease is a peer-to-peer asset leasing platform that allows users to list and lease assets such as:

- Houses and apartments
- Vehicles
- Electronics
- Tools and equipment
- Furniture
- Any rentable asset

## Features

### Authentication
- User Registration
- User Login
- Session Management

### Listings
- Create Listings
- View Listings
- Edit Listings
- Delete Listings

## Tech Stack

- Node.js
- Express.js
- SQLite
- EJS
- HTML
- CSS
- JavaScript
